First, make sure that the native-electron app is running on local machine, on the default port of 9090, to receive the control connection from this web app. Then click the Start button to start drawing, and double click anywhere to stop. Click on the color box to change the pen color, before starting to draw. Click anywhere and drag to draw. The drawing disappears 8s after completing the last continuous drawing. Use Shift key when completing a drawing to change the disappearance timer to 60s.
